# 📦 Smart Inventory Management System
### מערכת חכמה לניהול מלאי וציוד | מרכז הובלה

---

## 💡 About the Project | אודות הפרויקט
This system was built to solve the challenges of real-time inventory tracking in dynamic environments. It provides a lightweight, mobile-first, and highly intuitive interface to manage unit assets without the clutter of traditional spreadsheets.

המערכת פותחה במטרה להנגיש את ניהול המלאי ליחידות שטח, תוך דגש על מהירות, פשטות וחווית משתמש מודרנית (Mobile-First). היא מחליפה עבודה ידנית בקבצי אקסל במערכת ענן שמתעדכנת בזמן אמת.

---

## ✨ Key Features | יכולות מרכזיות

*   **📱 Mobile-First Design:** ממשק מותאם מלא לטלפונים ניידים - גישה מהירה לשטח.
*   **⚡ Real-Time Sync:** סנכרון מיידי בענן באמצעות Supabase.
*   **📸 Smart Media:** תמיכה בחיבור למצלמה ישירות מהטלפון להעלאת תמונות פריטים.
*   **🛡️ Anti-Duplication:** מנגנון חכם למניעת כפילויות – המערכת מזהה פריטים קיימים ומעדכנת כמויות במקום לשכפל.
*   **⚙️ Admin Control:** ניהול מלא (הוספה, עריכה, מחיקה) תחת הגנת סיסמה.
*   **📊 Visual Insights:** תצוגה גמישה בין רשימה (טבלה) לגלריית פריטים.

---

## 🚀 Future Scalability | פוטנציאל להתרחבות
This system is designed to be **modular and scalable**. It can be deployed in any unit or organization by simply updating the configuration keys.
המערכת תוכננה כ"תבנית" (Template). כל יחידה או חברה שתרצה לאמץ את המערכת תוכל לעשות זאת במינימום מאמץ טכני:

1.  **Configuration:** Swap the Supabase project URL and API Keys.
2.  **Branding:** Replace the logo file (`Logo_hovala_new.png`).
3.  **Deployment:** Deploy instantly via GitHub Pages or any static host.

## 🛠 Tech Stack | טכנולוגיות

*   **Database:** Supabase (PostgreSQL + Storage).
*   **Icons:** FontAwesome.
